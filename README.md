# 🏨 Hotel Management System (C++)

A comprehensive command-line application designed to streamline hotel operations. This system manages guest check-ins, room allocations, and billing, utilizing C++ file handling for reliable data persistence.

![C++](https://img.shields.io/badge/Language-C++-00599C?style=for-the-badge&logo=c%2B%2B)
![Industry](https://img.shields.io/badge/Type-Management%20System-orange?style=for-the-badge)

---

### ✨ Key Features
* 🔒 **Secure Login:** Administrative access control to protect guest data.
* 📝 **Guest Registration:** Efficiently record guest details and assign room numbers.
* 📜 **Database Management:** View and search through the complete list of registered guests.
* 💳 **Automated Billing:** Calculate stay duration and total costs during check-out.
* 📁 **Binary Data Persistence:** Stores information in `.dat` files using binary I/O, ensuring data is saved even after the program exits.
